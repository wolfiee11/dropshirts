# DropShirts.BD

The official repository for **DropShirts.BD**, a minimalist streetwear brand based in Narayanganj, Bangladesh. 

## About The Project
DropShirts.BD focuses on high-quality, heavyweight essential tees. This repository contains the source code for our storefront, built for speed, simplicity, and a premium aesthetic.

## Tech Stack
- **Frontend:** HTML5, CSS3
- **Hosting:** GitHub Pages
- **Design Philosophy:** Minimalist, Essential, Architectural

## Features
- ✅ Clean, responsive product showcase
- ✅ Optimized for mobile shopping
- ✅ Seamless bKash payment integration

## Current Status
- **Collection:** Drop 01
- **Focus:** Quality fabric & clean design
- **Location:** Narayanganj, BD
